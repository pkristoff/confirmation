Confirmation
================
v0.1
Features:
o devise implementation for candidate & admin
o Home page
o About page
o Admin:
    - login
    - creation
    - reset password
    - edit account
    - cancel account
    - list of Admins
o Candidate:
    - login
    - creation
    - reset password
    - edit account
    - cancel account
    - list of Candidates

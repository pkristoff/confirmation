This was created in an attempt to keep DB objects from being created to generate the sorting_candidate_selection pane.

Each instance represents a row in _sorting_candidate_selection.html.erb

Parameters:

  • :cand_info Candidate information shown in the sorting table.

  • :cand_event_info A hash that has the information necessary to generate the status of each candidate event for a candidate.

  • :candidate_event The candiate event information (verified, completion_date) being editied in mass_edit_candidates_event.html.erb
